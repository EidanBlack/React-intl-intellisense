# Intl Hover Helper for react-intl

Extensión para Visual Studio Code que mejora la experiencia al trabajar con `react-intl` en proyectos React, permitiéndote:

- 🪄 Ver la **traducción de una clave** al hacer _hover_ sobre `messages["clave"]`
- 🔍 Ir directamente a la definición en tu archivo de traducciones con `Ctrl + Click`
- ⚠️ Mostrar una advertencia si la clave no existe en el archivo de traducción
- 🔧 Configurar fácilmente la ruta de tus archivos de idioma

---

## 🛠️ Instalación

### Desde el Marketplace

1. Buscá la extensión por su nombre: **react-intl-intellisense**
2. Hacé clic en **Instalar**
3. ¡Listo!

### O desde archivo `.vsix`

Si recibiste el archivo `.vsix`:

```bash
code --install-extension react-intl-intellisense-0.0.1.vsix
```
